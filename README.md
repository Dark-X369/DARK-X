## INSTALLATION [TERMUX] :
                              
* `apt update`
* `apt install git -y`
* `pkg install git`
* `pkg install mpv -y`
* `git clone https://github.com/Dark-X369/DARK-X`
* `cd DARK-X`
* `chmod +x *`
* `bash install.sh`
* `exit`
